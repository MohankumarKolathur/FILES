MADAD

A Readability Annotation Tool for Arabic Text
At iWAN Research Group, I undertook the development of MADA, an annotation tool designed for annotating datasets in the field of Natural Language Processing (NLP). As a full-stack developer, I assumed complete end-to-end responsibility for this project as a single individual contributor. From scratch, I crafted the user interface (UI) to deliver a seamless and intuitive experience. I implemented document annotation features, allowing users to label and annotate specific sections within textual documents. Additionally, I developed token-based annotation capabilities to mark and classify individual tokens for fine-grained analysis. By integrating schema-based annotation, I enabled users to annotate data based on predefined guidelines, ensuring standardized annotations across datasets. Through my contributions, MADA empowers researchers and practitioners in NLP with efficient dataset annotation and analysis.

Nokia VAMS

Vulnerability Assessment & Management System
I played a vital role as a full-stack developer in the development of Nokia VAMS, a robust Vulnerability Assessment and Management System. This comprehensive system empowered security experts to assess, mitigate, and manage vulnerabilities across over 500 Nokia products at different version levels. Leading the front-end development, I designed and implemented a framework for the user interface (UI), optimizing developer productivity and ensuring consistency. Additionally, I developed a parsing system to efficiently process vulnerability reports from multiple scanners, enabling quick analysis and remediation. By creating connectors/interfaces for various scanners and Fault Management tools, I facilitated seamless integration for report retrieval. To maintain real-time visibility, I implemented a scheduler that periodically updated the status of each vulnerability. Through my contributions, Nokia VAMS provided a powerful solution for comprehensive vulnerability assessment and management across a diverse range of Nokia products.
